shedOS

shedOS is the operating system I'm currently working on. It is written in C and uses a custom UEFI bootloader. Already implemented functions include basic things like loading, setting up an identity page map and registering interrupt handlers. I mainly follow random articles from wiki.osdev.org and the YouTube tutorial series How to Make an OS from Poncho which I can highly recommend. There is still much to learn, so the codebase will certainly change a lot. My main goal is education and learning how things are done on the low level, not necessarily creating a usable kernel. Once I get further and manage to implement a proper userspace, I will add some useful tools, games and maybe port existing software to add at least some usability.
